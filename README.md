# موقع شخصي احترافي

موقع شخصي احترافي مع لوحة تحكم مدمجة مع Firebase.

## المميزات
- تصميم عصري ومتجاوب
- لوحة تحكم كاملة
- نظام مصادقة آمن
- إدارة المشاريع والمحتوى
- دعم كامل للغة العربية
- واجهة مستخدم سهلة الاستخدام

## التقنيات المستخدمة
- HTML5 & CSS3
- JavaScript (ES6+)
- Firebase (Auth & Firestore)
- Firebase Storage
- Firebase Hosting
